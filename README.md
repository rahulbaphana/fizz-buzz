#Fizz-Buzz:
#### When number is divisible by 3, print “fizz”
#### When number is divisible by 5, print “buzz”
#### When number is divisible by 3 and 5, print “fizzbuzz”

## Example:
#### Input : ```List(1, 2, 3, 5, 15,12, 11)```
#### Output : ```List(“1”, “2”, “fizz”, “buzz”, “fizzbuzz”, “fizz”, “11”)```
